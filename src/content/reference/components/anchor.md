`<anchor>` is a special button component that opens the given link when clicked.

<Sandpack>

```js App.js
export default function App() {
  return <anchor url="https://github.com/ReactUnity"
    style={{ margin: 20 }}>
    Go to GitHub
  </anchor>;
};
```

</Sandpack>

Affects the pivot point for `scale` and `rotate`. By default it is at the center.

### Valid values

- Positional keyword: `top`, `right`, `bottom`, `left`, `center`, `top left` etc.
- Pixel value: `50px 20px`
- Percentage values: `50% 20%`, `40px 50%` etc.

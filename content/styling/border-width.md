jsx: borderWidth

animatable: true
code: style/border-width
---

Changes the border width of all sides of the element's rectangle.

Alternatively, each side can be set separately with the following properties:

- `border-top-width`
- `border-right-width`
- `border-bottom-width`
- `border-left-width`

### Valid values

- Pixel value: `0`, `5px`, `20px` etc.

React Unity

Simple Container Component

view is the simplest container component. It can be used to encapsulate any number of children.

Code (jsx)
function Example() {
  return <view style={{
    padding: 20,
    backgroundColor: 'crimson',
    color: 'white',
  }}>
    Hello world
  </view>;
};


ReactUnityRenderer.render(<Example />);
Result

Scrollable Container Component

scroll is used to create a scrollable component.

Code (jsx)
function Example() {
  return <scroll style={{
    backgroundColor: 'white',
    padding: 20,
    height: 160,
  }}>
    Lorem ipsum dolor sit amet consectetur adipisicing elit. Amet molestiae officia sit numquam! Distinctio mollitia deserunt commodi iusto quam hic, in velit perspiciatis! Quaerat pariatur non, esse quam voluptatibus incidunt.
  </scroll>;
};


ReactUnityRenderer.render(<Example />);
Result

Text Component

text is used to create a component whose only content is text. It is useful for creating interpolated text and being able to style text nodes.

Code (jsx)
function Example({ name }) {
  return <text style={{ color: 'yellow' }}>
    Hello {name}
  </text>;
};


ReactUnityRenderer.render(
  <Example name={'John Doe'} />
);
Result

Button Component

button can be used to create clickable components.

Code (jsx)
function Example() {
  const light = Globals.Light;


  return <button
    style={{ margin: 20 }}
    onClick={() => light.enabled = !light.enabled}>
    Toggle Light
  </button>;
};


ReactUnityRenderer.render(<Example />);
Result

Input Component

input creates a text input component.

Properties

onChange
An event fired when string in the input is changed by the user
Code (jsx)
function Example({ defaultValue }) {
  return <input
    value={defaultValue}
    style={{ margin: 20 }}
    onChange={val => console.log(val)} />;
};


ReactUnityRenderer.render(
  <Example defaultValue="This is default" />
);
Result

Toggle Component

toggle creates a boolean toggle component.

Properties

value
Initial value of the toggle.
onChange
Event to call when the value changes.
Code (jsx)
function Example() {
  const light = Globals.Light;


  return <view style={{
    flexDirection: 'row',
    backgroundColor: 'white',
    justifyContent: 'center',
  }}>
    <toggle
      onChange={val => light.enabled = val}
      value={light.enabled} />


    Toggle light
  </view>;
};


ReactUnityRenderer.render(<Example />);
Result

Image Component

image creates an image component. Alternatively, rawimage creates a raw image component.

Properties

source
Source of the image. Can be a url or the Texture2D object.
fit
Determines how to position the image inside the element.
Code (jsx)
const imageAddress = "https://avatars.githubusercontent.com/u/62178684";


function Example() {
  return <image
    style={{ height: 200 }}
    fit={2}
    source={imageAddress} />;
};


ReactUnityRenderer.render(<Example />);
Result
